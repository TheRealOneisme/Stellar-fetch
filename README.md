# Stellar Fetch

A command-line tool to download and explore NASA's Astronomy Picture of the Day (APOD) with ease.

## Overview

Stellar Fetch connects to NASA's official APOD API to fetch beautiful space imagery and detailed astronomical explanations. Each day, NASA releases a new high-resolution image of the cosmos—Stellar Fetch makes it simple to collect, organize, and explore them all.

## Features

- **Daily Downloads** - Fetch today's APOD or any date in history
- **Batch Operations** - Download entire date ranges automatically
- **Beautiful CLI** - Rich terminal interface with progress tracking
- **Offline Gallery** - Generate shareable HTML galleries
- **Smart Organization** - Auto-organize images by date
- **High Resolution** - Download full HD images from NASA servers

## Quick Start

```bash
pip install -r requirements.txt
python stellar_fetch.py
```

## Requirements

- Python 3.7+
- Internet connection
- NASA API Key (or use free DEMO_KEY)
